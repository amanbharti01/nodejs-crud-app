# Node.js CRUD Application (OpenShift Deployment)

This repository contains the OpenShift manifests required to deploy a Node.js CRUD application on Red Hat OpenShift.  
The application is built using Source-to-Image (S2I) and includes full configuration for:

- Deployment  
- Service  
- Route  
- ImageStream  
- BuildConfig
